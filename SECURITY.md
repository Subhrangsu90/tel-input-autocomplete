# Security Policy

## Supported Versions

Only the latest version of `ng-tel-input-autocomplete` is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0.0 | :white_check_mark: |
| < 1.0.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by opening a draft security advisory on GitHub or by contacting the maintainers directly.

Please do not report security vulnerabilities through public GitHub issues.

We will acknowledge your report within 48 hours and provide a detailed response within 7 days, including our verification results and proposed mitigation/patch plan.
